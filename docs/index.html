StampVault WordPress Plugin

A powerful plugin to manage and showcase your stamp collection in WordPress.

Features

  • Custom Post Type for Stamps
  • Taxonomies: Stamp Types, Countries, Themes
  • Admin columns and filters for easy management
  • Easy integration and extensibility

Installation

  1. Download or clone the repository.
  2. Copy the StampVault folder to your wp-content/plugins directory.
  3. Activate the plugin from the WordPress admin dashboard.

Usage

  • Add new stamps from the "Stamps" menu in the admin dashboard.
  • Assign Stamp Types, Countries, and Themes to each stamp.
  • Use the admin filters and columns to manage your collection efficiently.
